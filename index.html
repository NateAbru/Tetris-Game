GAME RULES AND INSTRUCTIONS

Align the different blocks as they drop down to complete rows

Continue completing rows to erase rows and score more points, reaching new levels.

Each new level speeds up the droping time for each block.

Race against the drop time to prevent the blocks from piling up to the top of the arena

CONTROLS

UP ARROW KEY = ROTATE

LEFT ARROW KEY = MOVE LEFT

RIGHT ARROW KEY = MOVE RIGHT

DOWN ARROW KEY = MOVE DOWN

SPACE BAR KEY = DROP

Z KEY = HOLD CURRENT SHAPE

SCORING

scoring is based on the below for every block that lands

Level 1: +37 + ((100 per row completed) x multiplier)

Level 2: +53 + ((125 per row completed) x multiplier)

Level 3: +78 + ((150 per row completed) x multiplier)

Level 4: +105 + ((200 per row completed) x multiplier)

Level 5: +139 + ((250 per row completed) x multiplier)

Level 6: +170 + ((350 per row completed) x multiplier)

Level 7: +250 + ((500 per row completed) x multiplier)

* Multiplier = # of consecutive rows completed at once
